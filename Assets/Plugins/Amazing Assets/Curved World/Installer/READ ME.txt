Curved World asset contains 3 packages for each render pipeline:

1) Built-in - package for built-in (default) render pipeline.
2) Universal - package for Universal scriptable render pipeline (URP).
2) High Definition - package for High Definition scriptable render pipeline (HDRP).

Based on project's render pipeline import one of the provided package.


This package is designed for:
• Unity version 2020.2
• Universal RP 10.2.2
• High Definition RP 10.2.2
